  * **Ubuntu 12.10 Quantal Quetzal:**
> > Download and compile the [source code](http://touchegg.googlecode.com/files/touchegg-1.1.1.tar.gz) following this [instructions](http://code.google.com/p/touchegg/wiki/CompileSourceCode)
> > Do you want a GUI to configure Touchégg? Check out [Touchégg-GCE](https://github.com/Raffarti/Touchegg-gce) developed by Raffarti
  * **Ubuntu 12.04 Precise Pangolin:**
> > Download and compile the [source code](http://touchegg.googlecode.com/files/touchegg-1.1.tar.gz) following this [instructions](http://code.google.com/p/touchegg/wiki/CompileSourceCode)
  * **Ubuntu 11.10 Oneiric Ocelot:** Touchégg is available in the Ubuntu official repository, to install it:
> > ` $ sudo apt-get install touchegg `
  * **Ubuntu 11.04 Natty Narwhal:**
> > Is necessary update uTouch to the last version adding this PPA:<br />
> > https://launchpad.net/~utouch-team/+archive/daily <br />
> > And install the DEB package from the [Launchpad page](https://launchpad.net/ubuntu/oneiric/+source/touchegg/1.0-0ubuntu1) or download and compile the source code: <br />
> > - [i386 DEB package](http://launchpadlibrarian.net/80526375/touchegg_1.0-0ubuntu1_i386.deb) <br />
> > - [amd64 DEB package](http://launchpadlibrarian.net/80528329/touchegg_1.0-0ubuntu1_amd64.deb) <br />
> > - [armel DEB package](http://launchpadlibrarian.net/80527615/touchegg_1.0-0ubuntu1_armel.deb) <br />
> > - [powerpc DEB package](http://launchpadlibrarian.net/80527664/touchegg_1.0-0ubuntu1_powerpc.deb) <br />
> > - [Source code](http://touchegg.googlecode.com/files/touchegg-1.0.tar.gz)
  * **Arch Linux:** Available in [AUR](https://aur.archlinux.org/packages.php?ID=46809) (thanks to Chenxiaolong)
> > `$ yaourt -S touchegg`
  * **Fedora and Mageia:** Available at http://pkgs.org/download/touchegg
> > If you have a MacBook follow this instructions: http://code.google.com/p/touchegg/wiki/Fedora17MacbookAirHOWTO
  * For other distros you can download and compile the **[source code](https://code.google.com/p/touchegg/downloads/list)**
